# Config Profiles

Curated CLI profile presets for first-success builds.

Use:

```bash
python tools/configure.py --list-profiles
python tools/configure.py --quickstart esp32_c6_quickstart
```

You can also save/load custom profiles:

```bash
python tools/configure.py --profile-save my_profile
python tools/configure.py --profile-load my_profile
```
